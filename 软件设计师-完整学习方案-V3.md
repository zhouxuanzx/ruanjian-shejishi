```

SQL：SELECT 列 FROM 表 WHERE 条件 GROUP BY 分组 ORDER BY 排序
事务ACID：原子性 一致性 隔离性 持久性
```

### 1-4 页式管理计算 ★新增考点★

```
一、逻辑地址结构：
┌──────────────┬──────────────┐
│   页号 P     │ 页内偏移 W   │
└──────────────┴──────────────┘
页号P = 逻辑地址 ÷ 页面大小（整除）
偏移W = 逻辑地址 % 页面大小（取余）

二、物理地址计算：
物理地址 = 物理块号 × 页面大小 + 偏移量

三、真题型计算：
已知：页面大小4KB(4096B)，逻辑地址8644
→ 页号 = 8644/4096 = 2
→ 偏移 = 8644%4096 = 452
→ 若页表：页号2→块号7，则物理地址 = 7×4096+452 = 29124

四、地址位数计算：
逻辑空间4GB = 2^32 → 32位地址
页大小4KB = 2^12 → 偏移占12位
页号 = 32-12 = 20位
```

### 1-5 数据结构与算法（8-10分）

```
线性表：
  顺序表→随机访问快O(1)，插入删除慢O(n)
  链表→随机访问慢O(n)，插入删除快O(1)
  栈→后进先出(LIFO)    队列→先进先出(FIFO)

二叉树遍历：
  前序：根左右  中序：左根右  后序：左右根  层次：从上到下

排序算法（时间复杂度+稳定性）：
  O(n²)：冒泡√  插入√  选择✗
  O(nlogn)：快速✗  归并√  堆✗
  O(n)：基数√
  √=稳定  ✗=不稳定
  稳定排序只有：冒泡、插入、归并、基数

查找：
  顺序查找O(n)  二分查找O(logn)(要求有序)
```

### 1-6 链表操作 ★新增考点★

```
单链表结构：
[数据|next] → [数据|next] → [数据|NULL]
  head                          tail

插入操作（在p后插入s）：
  s.next = p.next    ← 先接后面！顺序不能反
  p.next = s         ← 再接前面

删除操作（删除p后的结点q）：
  p.next = q.next    ← 跳过q

遍历链表：
  p = head
  while p != NULL:
      处理p.data
      p = p.next

查找结点：
  p = head
  while p != NULL && p.data != target:
      p = p.next

顺序表 vs 链表：
┌──────┬──────┬──────┐
│      │顺序表│ 链表 │
├──────┼──────┼──────┤
│随机访问│ O(1) │ O(n) │
│插入删除│ O(n) │ O(1) │
│二分查找│  能  │ 不能 │
└──────┴──────┴──────┘
```

### 4-2A 背包算法详解 ★新增考点★

```
0/1背包问题：
  有n个物品，重量w[i]，价值v[i]，背包容量W
  每个物品要么装(1)要么不装(0)，求最大价值

状态定义：dp[i][j] = 前i个物品、容量j时的最大总价值

状态转移（★考试填空高频）：
  if (j < w[i])
      dp[i][j] = dp[i-1][j];              // 装不下
  else
      dp[i][j] = max(dp[i-1][j],          // 不装第i个
                     dp[i-1][j-w[i]] + v[i]); // 装第i个

代码模板（考试填空重点位置用★★标注）：

for (i = 1; i <= n; i++) {            ★★ 循环边界
    for (j = 1; j <= W; j++) {        ★★ 循环边界
        if (j < w[i])                  ★★ 判断条件
            dp[i][j] = dp[i-1][j];    ★★ 装不下
        else
            dp[i][j] = max(           ★★ 取较大值
                dp[i-1][j],           // 不装
                dp[i-1][j-w[i]] + v[i] // 装
            );
    }
}
最终答案：dp[n][W]

考试填空高频位置：
  ① 循环条件 i<=n 或 j<=W
  ② 判断 j<w[i]
  ③ dp[i-1][j-w[i]] + v[i]
  ④ dp[n][W]（最终结果）
```

### 5-2A 类复制 / 原型模式 ★新增考点★

```
考试场景：实现一个"商品复制类"，既能复制又能复用公共属性

核心知识点：
1. Cloneable接口→实现复制功能
2. clone()方法→创建并返回副本
3. 构造方法→初始化属性
4. super()→调用父类构造

考试填空模板：

class Product implements Cloneable {
    private String name;
    private double price;

    public Product(String name, double price) {
        this.name = name;          ← 填空1：构造函数赋值
        this.price = price;
    }

    public Product clone() {      ← 填空2：实现clone方法
        return new Product(this.name, this.price);
    }
}

// 子类继承并扩展
class FoodProduct extends Product {
    private Date expireDate;

    public FoodProduct(String name, double price, Date d) {
        super(name, price);       ← 填空3：调用父类构造
        this.expireDate = d;
    }

    public FoodProduct clone() {
        return new FoodProduct(   ← 填空4：返回子类对象
            this.getName(), this.getPrice(), this.expireDate
        );
    }
}

高频填空位：
  ① this.xxx = xxx（构造赋值）
  ② super(xxx)（调父类构造）
  ③ implements Cloneable / extends ParentClass
  ④ return new XXX(...)
```
